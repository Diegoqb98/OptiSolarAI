# Modelos Entrenados

Este directorio almacena los modelos de Machine Learning entrenados.

## Modelos Previstos

1. **`price_predictor.pkl`**: Modelo para predecir precios horarios de electricidad
2. **`solar_predictor.pkl`**: Modelo para predecir producción solar

## Formato

Los modelos se guardan en formato `.pkl` o `.joblib` usando scikit-learn.

## Nota

Los archivos de modelos entrenados no se suben al repositorio debido a su tamaño.
Para entrenar los modelos, ejecutar los scripts correspondientes en `/src/models/`.
